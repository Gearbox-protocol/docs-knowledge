## Where to verify addresses

Always rely on the official deployments list for the canonical contract addresses. The list is maintained by Gearbox and reflects the current production deployments on supported networks.

Notes:
Do not copy addresses from third‑party sites or random posts. If an address in the UI doesn’t match the official list, stop and re‑check the URL and network before proceeding.

## How to self‑check (quick procedure)

Verifying an address takes less than a minute and prevents most phishing mistakes.

### Step 1 — Copy from the UI
Open the relevant screen (e.g., a pool, a Credit Account contract link, or a settings/info panel) and copy the contract address provided by the app.

### Step 2 — Open in a block explorer
Paste the address into the block explorer for the same network (for example, Etherscan for Ethereum). Check the network at the top of the explorer. Confirm the contract is verified and the page shows a contract, not an EOA, unless the context is an EOA by design (such as a multisig).
VERIFY: network in explorer matches the network shown in the UI

### Step 3 — Compare with the official list
Open the official deployments list and find the relevant item. Compare the full address (not a shortened preview) and the network. If both match, proceed. If they don’t, stop and confirm you are on the official UI domain and viewing the right network.
VERIFY: UI address equals the address in the official list; network matches

### Step 4 — Proceed with the action
Return to the app and continue with your action (deposit, swap, open/close account) once verification is complete. If any doubt remains, disconnect your wallet and verify the site URL and certificates. Bookmark the official site to reduce the chance of phishing in the future.

---

## Practical tips

Prefer links surfaced by the app to open explorers — they usually point to the correct network automatically. If the UI shows a mismatch warning, do not proceed; re‑check the network and the address immediately. When sharing an address with teammates, include the network and an explorer link.
