Все надо делать в 5 Thiniking режиме со включенным агентом. Для каждой главы лучше использовать отдельный чат (в противном случае у меня агента хуже работал).

Нужно загрузить файлы rules_for_model.md, structure.md, README.md и файлы по одному из папки needs_to_be_processed.

------------------
Промпт:

"Ты - агент. Строго следуй правилам для агента. Обработай главу N.
Ссылка на App: http://draft.client-v3.pages.dev/?dev-mode=false.
По итогу выдай md файл."
------------------

Выдается как правило файл, который нельзя скачать в md формате, поэтому можно написать второй промпт: "Выдай мне MD файл для копирования".
