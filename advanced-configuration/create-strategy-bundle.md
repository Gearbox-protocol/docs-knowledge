# Create Strategy Bundle

### What's a bundle?

Product perspective:

* Gearbox contracts configuration, allowing users to execute 1-click leveraged strategies.
* Set of adapters providing enough liquidity sources to execute swaps and liquidations

Technical perspective:

* Market with all the required tokens added with some quota and price feeds
  * Target assets (tokens actually held by user on CA and contributing to its Account Value) should have nonzero quota and adequate price feeds.
